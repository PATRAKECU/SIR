Sistema de Pronóstico de Propagación de Enfermedades Infecciosas – Modelo SIR

Este sistema permite simular la propagación de enfermedades infecciosas utilizando el modelo epidemiológico SIR (Susceptible – Infectado – Recuperado).

Modelo SIR

El modelo SIR describe la dinámica de transmisión mediante el siguiente sistema de ecuaciones diferenciales: 

        dS/dt = -βSI
        dI/dt = βSI - γI
        dR/dt = γI

Donde:

  • S: población susceptible
  • I: población infectada
  • R: población recuperada
  • β: tasa de contagio
  • γ: tasa de recuperación
